# Requirements (minimum)

- (Built on) Centos 7.2
- node 6.9.1
- devkit 3.1.3
- chrome 54.0.2840.100
- docker 1.10.3

# Commands

```
// Run for command list
make
```

```
// Launch application in development mode
make serve
```

```
// devkit release, build and run docker image
make build
// nginx will serve app on localhost:8080
```

```
// Lint code base
make test-lint
```
